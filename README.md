## Overview

Web Shop is a modern web application that allows users to browse and purchase products online. It is built using React with skapa componnent and provides a seamless shopping experience.

## Features

- Browse products by category
- Search for products
- Add products to cart
- View cart details
- Checkout process

## Installation

To get started with the project, follow these steps:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/im-priyan-04/web-shop
    cd web-shop
    ```

2. **Install dependencies:**

    ```bash
    npm install
    ```

3. **Start the development server:**

    ```bash
    npm start
    ```

4. **Open your browser and navigate to:**

    ```
    http://localhost:3000
    ```

## Usage

### Browsing Products

- Navigate to the home page to see a list of available products.
- Use the category filter to narrow down the products.

### Searching for Products

- Use the search bar at the top to find specific products.

### Adding to Cart

- Click on the "Add to Cart" button to add a product to your cart.

### Viewing Cart

- Click on the cart icon to view the items in your cart.

### Checkout

- Proceed to checkout by clicking the "Checkout" button in the cart.
